## What is HyLay?

HyLay lets you build game overlay UIs using any web framework, React, Vue, Svelte, or plain HTML/CSS/JS, and render them directly on top of a running Windows game.

It works by injecting a DLL into the game process that hosts a transparent WebView2 window over the game surface, and exposes a bidirectional event bridge so your UI can communicate with the game client in real time.

Think of it as **NUI for any game**, similar to how FiveM's NUI lets you build Lua-to-web bridges, or how NoesisGUI lets engines render XAML

## Key features

  <Card title="Framework agnostic" icon="layer-group">

## Designed for

- **Hytale plugins** — build HUD elements, menus, and overlays controlled by server-side Kotlin logic
- **Rapid UI iteration** — use your existing web tooling (Vite, hot reload, Tailwind) to build game UIs

## Not designed for

- Anti-cheat protected games (DLL injection will trigger detection)
- Non-Windows platforms (Windows 10+ required for WebView2)

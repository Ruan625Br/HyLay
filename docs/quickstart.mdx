## Step 1 — Create your UI

HyLay works with any web framework. The fastest way to start is with Vite + React:

```bash
npm create vite@latest my-overlay -- --template react-ts
cd my-overlay
npm install
```

Copy the HyLay JS bridge into your project:

```bash
# Copy from the hylay repo
cp hylay/bridge/useHyBridge.ts src/hooks/useHyBridge.ts
cp hylay/bridge/hy-bridge.d.ts src/types/hy-bridge.d.ts
```

---

## Step 2 — Use the bridge in your UI

```tsx
import { useHyEvent, useHyBridge } from "./hooks/useHyBridge"

function App() {
  const { setClickThrough } = useHyBridge()
  const [playerName, setPlayerName] = useState("—")

  useHyEvent("playerConnect", (payload) => {
    const data = payload as { playerName: string }
    setPlayerName(data.playerName)
  })

  return (
    <button
      onMouseEnter={() => setClickThrough(false)}
      onMouseLeave={() => setClickThrough(true)}
    >
      Hello, {playerName}
    </button>
  )
}
```

---

## Step 3 — Run your UI dev server

```bash
npm run dev
# Vite starts on http://localhost:5173
```

HyLay loads `http://localhost:5173` by default. In production you'll point it at a built bundle.

---


## Step 4 — Send events from the client

From your Kotlin plugin or any process that can write to a Named Pipe:

```kotlin
HyLayBridge.emitEvent("playerConnect", mapOf("playerName" to "Ruan625Br"))
```

Your React component will receive the event and update in real time.
